# FinOps hubs automation

The FinOps hubs PowerShell commands help you deploy and manage FinOps hubs instances.

<br>

## Commands

- [Deploy-FinOpsHub](Deploy-FinOpsHub.md) ‚Äì Deploy your first hub or update to the latest version.
- [Get-FinOpsHub](Get-FinOpsHub.md) ‚Äì Get details about your FinOps hub instance.
- [Initialize-FinOpsHubDeployment](Initialize-FinOpsHubDeployment.md) ‚Äì Initializes the deployment for FinOps hubs.
- [Register-FinOpsHubProviders](Register-FinOpsHubProviders.md) ‚Äì Registers resource providers for FinOps hubs.
- [Remove-FinOpsHub](Remove-FinOpsHub.md) ‚Äì Deletes a FinOps hub instance.
